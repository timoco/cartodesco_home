OpenGeo is now Boundless

Open source software has come to dominate much of the IT world. As Boundless, we ensure that the geospatial community reaps the tremendous benefits of open source. We package the innovation, quality, and power of software created in an open manner to create geospatial applications of greater scale and speed at lower costs.

We are the same company you’ve known for many years — our mission remains to develop and commercially maintain the best open source software for geospatial applications — but we are now stronger and more capable than ever.

The future is not just open. It is Boundless.

OpenGeo Suite

OpenGeo Suite is a complete geospatial platform built on powerful open source software that leverages standards-based technology to take advantage of the web.

Features

  • Scalable data storage with PostGIS
  • Easy integration to existing spatial data sources with GeoServer
  • High performance web map tile delivery with GeoServer and GeoWebCache
  • Flexible user interfaces with OpenLayers
  • Desktop data management and analysis with QGIS
  • Integrated performance monitoring and metrics with Mapmeter
  • Leading open standards support from top to bottom
